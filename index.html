Structurateur de Prompts pour Applications Pédagogiques

Aider les enseignants à générer, grâce à l'IA, des applications éducatives / pédagogiques sur mesure avec Claude AI en structurant les prompts de manière professionnelle

1

Identité de l'IA

? Définissez le rôle que Claude doit incarner. Par exemple : "Expert en codage web et professeur de mathématiques".
2

Contexte & Public cible

? Sélectionnez le niveau scolaire visé / le public visé pour adapter le vocabulaire et la complexité du contenu.
3

Objectif Principal

? Décrivez précisément ce que l'application doit permettre de faire. Soyez spécifique sur les compétences visées.
4

Structure de l'Application

? Décrivez l'organisation de l'interface : nombre de pages, zones de saisie, boutons, onglets, etc.
5

Contenu & Actions

? Détaillez précisément chaque élément de l'application : types d'exercices, feedbacks, animations, interactions, etc.
6

Format de Sortie

? Pour une application web interactive, HTML est le format recommandé. React JSX pour des applications plus complexes.
7

Base de Données / Référence

? Optionnel : Fournissez un lien vers un Google Sheets, CSV ou autre fichier contenant les données à utiliser (questions, réponses, etc.).
8

Publication en Ligne

? Choisissez la plateforme pour héberger votre application. GitHub Pages est gratuit et facile à utiliser.

⚖️ Conformité RGPD :
Les applications pédagogiques créées avec ce générateur doivent respecter le Règlement Général sur la Protection des Données (RGPD) français et européen. Veillez à :
  • Ne collecter que les données strictement nécessaires au fonctionnement pédagogique
  • Informer les utilisateurs (élèves et parents) de l'utilisation de leurs données
  • Obtenir le consentement parental pour les mineurs de moins de 15 ans
  • Assurer la sécurité et la confidentialité des données collectées
  • Permettre l'accès, la rectification et la suppression des données personnelles
  • Ne pas transférer de données personnelles en dehors de l'UE sans garanties appropriées

Pour plus d'informations : CNIL.fr
